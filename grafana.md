## start.sh
``` bash
#!/bin/bash
docker run -d \
-p 3000:3000 \
--name=grafana \
-e "GF_INSTALL_PLUGINS=alexanderzobnin-zabbix-app" \
-v /opt/grafana/etc:/etc/grafana -v /opt/grafana/var:/var/lib/grafana \
grafana/grafana

```

## 修改Home
``` bash
#root权限进入
docker exec -it --user root grafana /bin/bash
#备份home.json
```
